Privacy Policy:

1. Data Sources and Types

The App uses data from the Overpass API for a slope analysis, from opensnowmap.org for the map and the Flutter Geocode Library for location determination. Nils Knittel does not collect personal data from users.

2. Data Usage and Disclosure

The data obtained from the Overpass API, opensnowmap.org and the Flutter Geocode Library is solely for the purpose of location determination in the App. Nils Knittel has no control over the data collection and processing by these external sources.

4. Changes to the Privacy Policy

Nils Knittel reserves the right to update this Privacy Policy. Changes will be published within the App.
